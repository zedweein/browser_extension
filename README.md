

# browser_extension



## Usage

1.打开浏览器-设置-更多-扩展程序
2.加载已解压扩展程序 
3.在需要抓包的界面,输入相关参数
4.点击开始抓包 

## Developing

目前使用的是google devtool来获取response,可以使用原生API获取更多关于HTTP请求相关内容
